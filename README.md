# Banking System in Python

A simple command-line banking system built using Python. This project allows users to create accounts, deposit money, withdraw money, and check account balances.

## Features

* Create a new bank account
* Deposit money
* Withdraw money
* Check account balance
* Input validation
* Prevents overdrawing an account

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/banking-system.git
```

2. Navigate to the project folder:

```bash
cd banking-system
```

3. Run the program:

```bash
python prawwal.py
```
